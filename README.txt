Інструкція розгортання на GitHub Pages (мінімум кроків):
1. Створіть GitHub-репозиторій (публічний) з назвою на ваш вибір.
2. Завантажте всі файли з архіву в корінь репозиторія (через веб-інтерфейс або git).
3. У налаштуваннях репозиторія (Settings) увімкніть Pages (Branch: main / root).
4. Через кілька хвилин сайт буде доступний за адресою: https://<your-username>.github.io/<repo>/

Щоб додати новину:
1. Завантажте PDF/DOCX у папку /news.
2. Додайте запис у news.json у форматі:
   { "title": "Заголовок", "file": "news/filename.pdf", "date":"YYYY-MM-DD" }
3. Збережіть зміни — новина з'явиться автоматично.

Примітка: файли .pdf у цьому архіві — заглушки. Замініть їх на реальні документи.
